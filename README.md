# お金の育て方 🌱

未来の資産を「見える化」する、やさしい複利シミュレーター

## 概要
お金の育て方は、元本・利回り・運用年数を入力するだけで、 <br />
資産がどのように増えていくかをグラフと表で直感的に確認できる<br />
複利シミュレーションツールです。

投資初心者の方でも「将来どれくらい増えるのか」を<br />
感覚的に理解できることを目的に制作しました。

## 主な機能
- 元本・年利・運用年数の入力による複利計算
- 期間を分けたシミュレーション（途中で利回りや年数を変更可能）
- Chart.js を用いた資産成長グラフの表示
- 各年ごとの資産推移を一覧で確認できるテーブル表示
- 入力値チェック（未入力・不正値のエラーメッセージ表示）


## こだわったポイント
- **初心者向けUI**  
  専門用語をできるだけ使わず、説明文を添えて安心して入力できる設計にしました。
- **視覚的な理解**  
  数字だけでなくグラフで成長を確認できるようにし、  
  「増えていく実感」を得られることを意識しました。
- **段階的な運用設定**  
  市場の暴落や相場環境の変化を想定し、  
  途中で利回りや運用期間を変更できるよう、複数条件を追加できる仕様にしています。
